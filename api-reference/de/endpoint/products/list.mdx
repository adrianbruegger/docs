title: 'Produkte auflisten'
description: 'Produkte Ihrer Organisation abrufen oder durchsuchen'
openapi: 'GET /products'
version: 'Deutsch'
---

Ohne Abfrageparameter `q` liefert der Endpunkt die vollständige Produktliste. Mit `q` und einem nicht leeren Suchbegriff erfolgt eine Volltextsuche (z. B. über Name und Barcode). Die Parameter `limit` (1–250, Standard 100), `offset` (0–10 000, Standard 0) und `store` wirken nur bei einer Suche mit `q`; `store` begrenzt die Treffer auf Produkte, die für diesen Laden freigegeben sind.

Antworten enthalten **`meta`** (`total`, `returned`, bei Verwendung von `q` außerdem `limit` und `offset`). Siehe [Einführung](/api-reference/de/introduction#listen-antworten-meta).
