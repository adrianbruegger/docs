title: 'Kategorien auflisten'
description: 'Alle Kategorien Ihrer Organisation abrufen'
openapi: 'GET /categories'
version: 'Deutsch'
---

Optional **`limit`** (1–5000, Standard 100) begrenzt die Anzahl der Kategorien. Die Antwort-**`meta`** enthält `total`, `returned` und `limit`. Siehe [Einführung](/api-reference/de/introduction#listen-antworten-meta).
