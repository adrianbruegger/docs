---
title: 'List Products'
description: 'List or search products in your organization'
openapi: 'GET /products'
version: 'English'
---

Without the `q` query parameter, the endpoint returns the full product list. With `q` set to a non-empty search string, results are returned from full-text search (for example on name and barcode). The `limit` (1–250, default 100), `offset` (0–10000, default 0), and `store` parameters apply only when searching with `q`; `store` limits hits to products allowed for that store.

Responses include **`meta`** (`total`, `returned`, and when using `q`, `limit` and `offset`). See [Introduction](/api-reference/en/introduction#list-responses-meta).
