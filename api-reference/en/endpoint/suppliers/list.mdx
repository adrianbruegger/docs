---
title: 'List Suppliers'
description: 'Get all suppliers of your organization'
openapi: 'GET /suppliers'
version: 'English'
---

Use optional **`limit`** (1–5000, default 100) to cap how many suppliers are returned. The response **`meta`** includes `total`, `returned`, and `limit`. See [Introduction](/api-reference/en/introduction#list-responses-meta).
