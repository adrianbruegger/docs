---
title: 'List Orders'
description: 'Get all orders of your organization'
openapi: 'GET /orders'
version: 'English'
---

Use optional **`limit`** (1–5000, default 100) to cap how many orders are returned (newest first). The response **`meta`** includes `total`, `returned`, and `limit` (`total` and `returned` match the size of `data` in this response). See [Introduction](/api-reference/en/introduction#list-responses-meta).
