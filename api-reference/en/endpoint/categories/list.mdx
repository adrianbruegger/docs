---
title: 'List Categories'
description: 'Get all categories of your organization'
openapi: 'GET /categories'
version: 'English'
---

Use optional **`limit`** (1–5000, default 100) to cap how many categories are returned. The response **`meta`** includes `total`, `returned`, and `limit`. See [Introduction](/api-reference/en/introduction#list-responses-meta).
